# Legacy UI

This directory contains the legacy AWX PatternFly UI that ships with the upstream AWX codebase.

**The active Forail frontend is in a separate repository:** [forail-frontend](https://github.com/forail-platform/forail-frontend)

The legacy UI code is retained for backward compatibility with AWX internals (management commands, static file collection) but is not actively developed or used in production Forail deployments.

For frontend development, see the `forail-frontend` repository.
